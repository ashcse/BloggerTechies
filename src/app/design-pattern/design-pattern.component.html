Design patterns

Design patterns are the core of Object oriented programming. These are fundamental concepts, which when used appropriately helps building highly maintainable and extensible software designs. Any design pattern is a well proven solution to a specific type of design challenge. For example factory patterns help solving design challenges related to objects creations. There are many design patterns but here I would like to mention few important which I have used alot any software system will follow at least these patterns.

Following are the commonly used design patterns.
