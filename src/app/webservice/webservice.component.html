What is Web Service?

There are many definitions available over internet hence I would like to describe briefly based on my understanding.
"Web service is any software which is available to be used over internet and follows some protocals can be considered as a web service.""

SOAP or RESTful

Two commonly used types of web services are: SOAP and REST.

Basically these services are characterized by protocals they use. These protocals involve transmission protocals like TCP/IP, HTTP, UDP and messaging protocals like SOAP (XML based message) or JSON messages.
